## Architecture

This is an **Astro 5 static site** (output: `static`) deployed to GitHub Pages. It serves as a personal portfolio and Turkish-language technical blog.

### Configuration Flow

Site behavior is driven by **`src/config.yaml`** — blog pagination (9 posts/page), post permalink pattern (`/%slug%`), language (`tr`), theme (`system`), and analytics. A custom integration in `vendor/integration/` exposes this config as a virtual module (`astrowind:config`) so all components can import it.

`src/navigation.ts` defines header/footer navigation links.

### Content

Blog posts live in **`src/data/post/`** (~90 markdown files, primarily Turkish tech articles). Frontmatter format:

```yaml
publishDate: 2021-08-24T00:00:00Z
author: Hakan Çelik
title: "Post Title"
excerpt: "Short description"
category: Python
tags: [python, django]
image: ~/assets/images/blog/some-image.jpg
```

Slugs are auto-generated from filenames (Turkish characters handled via `limax`). Static markdown pages (`privacy.md`, `terms.md`) live directly in `src/pages/`.

### Routing

- `/` — Home (fetches GitHub stars + PyPI download stats at build time, cached 1h)
- `/:slug` — Individual blog post
- `/blog` and `/blog/[page]` — Paginated blog listing
- `/category/:cat/[page]` and `/tag/:tag/[page]` — Taxonomy pages
- `/rss.xml` — RSS feed (generated by `src/pages/rss.xml.ts`)

### Component Hierarchy

```
Layout.astro (root — ViewTransitions, lang/dir)
└── PageLayout.astro / LandingLayout.astro / MarkdownLayout.astro
    ├── components/widgets/   — Page sections (Hero, Features, Stats, BlogLatestPosts, etc.)
    ├── components/ui/        — Primitives (Button, Headline, ItemGrid, Timeline)
    ├── components/blog/      — Blog-specific (SinglePost, List, Grid, Pagination, Tags)
    └── components/common/    — Cross-cutting (Metadata, Analytics, Image, ToggleTheme)
```

### Utilities

- `src/utils/blog.ts` — Blog collection loading, filtering, pagination logic
- `src/utils/permalinks.ts` — URL/slug generation
- `src/utils/frontmatter.ts` — Custom remark/rehype plugins (reading time, responsive tables, lazy images)
- `src/utils/images.ts` / `images-optimization.ts` — Image helpers (uses `unpic` + `sharp`)

### Path Alias

`~/*` maps to `src/*` (configured in both `tsconfig.json` and `astro.config.ts`).

### Deployment

GitHub Actions (`.github/workflows/deploy.yml`) runs `npm ci && npm run build` on push to `main`, then deploys `dist/` to GitHub Pages. Site URL: `https://hakancelik.dev`.
